# dell_e6430_hiRes
Custom EFI files for dell latitude e6430 hiRes for macOS Mojave

Works:
- SD card reader
- Keyboard
- Keyboard Back
- Touchpad
- Trackpoint
- All usbs
- Hdmi
- Battery & indicator
- Headphone jack
- Screen
- Speakers
- changing volume

Doesn't work:
- VGA
- Webcam
- sometimes says that you have to replace the battery


If you want to use it, just replace them with provided files in EFI folder
